in-memory for local dev and tests.
> `Python` ┬╖ `Apache Kafka` ┬╖ `Spark Structured Streaming` ┬╖ `SQL` ┬╖ `DuckDB/Snowflake` ┬╖ `Power BI` ┬╖ `Docker`
